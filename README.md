## Description

Cette application web permet de visualiser les campings des environs.


## Installation

Pour pouvoir utiliser ce serveur web, veuillez suivre les étapes suivantes:
- Ouvrir un terminal et inscrire mongod
- Ouvrir un deuxième terminal et se rendre à la racine du projet
- Inscrire "npm install"
- Pour lancer le serveur, inscrire "npm start"
